# fast food card matching game
date: April 10 2026

game objective: match each card with a card that has the matching fast food item. Try to beat the game in the least amount of attempts as possible

game rules: If two cards are picked with different images then one tick is added to the attempts counter and the cards are hidden again. 
If two matching cards are found then the those cards are always seen and can no longer be picked.

tech used: bootstrap(navbar and cards), buttons, and animations.
